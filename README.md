# EasyLine

Lightweight MT5 drawing tool with OHLC snap, draggable toolbar, and baseline-based horizontal level expansion.

轻量级 MT5 划线工具，支持 OHLC 吸附、可拖动工具条，以及基于基准横线的扩展价格位吸附。
